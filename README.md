# Contact Management App - JavaScript - 2016
Add, modify, and delete contacts from your list.

### Features
1. Add, modify, and delete contacts
2. Contact description:
  - Avatar
  - First Name
  - Last Name
  - City
  - Country
  - Address
  - ZIP code
3. Avatar - select image
4. Multiple contacts
5. Alert messages

### Technologies: 
1. Angular JS
